# Panache Advanced Dental Care Website

A professional dental and maxillofacial surgery website for Dr. Sandeep Prakash, built with Flask and Bootstrap.

## Features

- **Responsive Design**: Mobile-friendly layout using Bootstrap 5
- **Professional Theme**: Custom red, brown, and white color scheme
- **Multiple Pages**: Home, About, Services, Skills, and Contact pages
- **Appointment Booking**: Interactive appointment booking form
- **Service Showcase**: Detailed display of medical, surgical, and dental services
- **Professional Profile**: Comprehensive presentation of doctor's expertise and qualifications

## Pages

1. **Home**: Hero section with doctor introduction and featured services
2. **About**: Professional background, education, and career timeline
3. **Services**: Complete list of medical, surgical, and dental services
4. **Skills**: Detailed showcase of surgical procedures and expertise
5. **Contact**: Clinic information, appointment booking, and contact details

## Installation & Setup

1. **Install Python Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Flask Application:**
   ```bash
   python app.py
   ```

3. **Access the Website:**
   Open your browser and go to `http://localhost:5000`
